# Sistema de Gestão de Reservas para Pousadas

## 🎓 Contexto Acadêmico
* **Disciplina:** Introdução à Programação
* **Semestre:** 1º Semestre
* **Objetivo:** Desenvolver um sistema de gestão em C, focando em manipulação de estruturas de dados, alocação dinâmica de memória e persistência de dados em arquivos CSV.

## 🚀 Sobre o Projeto
Sistema de linha de comando (CLI) voltado para a automação do fluxo de uma pousada. O software gerencia o registro de hóspedes, a ocupação de quartos e o histórico de reservas, garantindo que os dados persistam entre as execuções do programa.

## ✨ Funcionalidades
* **Gestão de Hóspedes:** Cadastro, edição e remoção de registros.
* **Controle de Reservas:** Verificação de disponibilidade e atribuição de quartos.
* **Persistência de Dados:** Salvamento automático das informações em arquivos CSV para consulta posterior.
* **CLI Interativo:** Navegação clara por menus para facilitar a operação do usuário.

## 🛠️ Tecnologias Utilizadas
* **Linguagem:** C
* **Manipulação de Arquivos:** Operações de I/O em arquivos (CSV)
* **Estrutura de Dados:** Structs e Ponteiros para alocação dinâmica
